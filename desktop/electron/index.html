Professor Demo Notes

This desktop build is optimized for stable teaching/demo workflows.

  • Stable focus: DSC/TGA analysis, compare workspace, save/load project archives, CSV/DOCX artifacts.
  • Preview modules (DTA, kinetics, deconvolution) are intentionally out of scope for demo promise.
  • Streamlit remains available as fallback/reference implementation without modification.
